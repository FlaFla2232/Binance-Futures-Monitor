Binance Futures WebSocket Monitor

Real-time Binance USDT-M Futures ticker monitor (WebSocket-only) with filters and signal generation.

Run locally

This is a Python/Flask app and cannot run on GitHub Pages. To use it:

1) python -m venv .venv
2) pip install -r requirements.txt
3) python binance_futures.py
4) Open http://127.0.0.1:5000
